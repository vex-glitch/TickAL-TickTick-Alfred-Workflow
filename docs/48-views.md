# Native TickTick views

_TickAL docs: [Home](00-index.md) · [Setup](30-setup.md) · [Cheatsheet](95-cheatsheet.md)_

> Views TickAL does not re-render - one keyword jumps you straight into the native screen inside TickTick

**Keywords:** `tha` · `tpo` · `tmx` · `tca` · `tst` · **Hotkeys:** (set in canvas)

## View keywords

| Keyword | Opens |
|---------|-------|
| `tha` | 🔄 Habits |
| `tpo` | 🍅 Pomodoro / Focus |
| `tmx` | 🧭 Eisenhower Matrix |
| `tca` | 📅 Calendar (day / week / month / year) |
| `tst` | 📊 Statistics |

These deep-link into TickTick and carry no in-Alfred chords or drill, unlike the [in-Alfred views](41-browse-drill.md), which render as lists you can act on and walk.
